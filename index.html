Player web: link y pasos para configurarlo
Link al player web:
https://debq2.debmedia.com/debsign/assets/debPlayerWeb/views/config.html
Pasos para configurar el player:
  1. Abrir el link.
  2. Si la pantalla que se muestra es similar a la siguiente, pasar al paso 4:
    La pantalla debe mostrar campos para "URL del Servidor", "ID de Pantalla", "Token" y opciones de configuración.
  3. Si la pantalla que se muestra no es similar a la anterior, presionar "Ctrl + Q".
    Se cambiará a la pantalla de configuración mencionada en el paso 2.
  4. Se deben completar los siguientes campos:
    • URL del servidor: Es la URL dónde está funcionando la compañía. Para el caso de las DEMOs comerciales la URL es:
      https://debq2.debmedia.com/debsign
    • Token: Corresponde al ID y token de cada pantalla con el formato "ID-token", y se podrá encontrar en la sección:
      "debSign → Panel de Compañía → Pantallas"
      El token tiene el formato (tomar el de usuarios Numia): ID-token (ejemplo: 4679-R8TbGe)
  5. Una vez completados los campos, presionar el botón "Guardar" para aplicar la configuración.
Nota: Los demás campos de configuración (impresora, teclado, mouse, etc.) son opcionales y pueden configurarse según las necesidades específicas de cada pantalla.
